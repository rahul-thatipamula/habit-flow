## 📦 How to Install

1. **Download the DMG**: Click the link above and download the `.dmg` file.
2. **Mount and Install**: Double-click the `.dmg` file and drag the **Habit Flow** icon into the **Applications** folder shortcut.
3. **Launch the App**: Open your **Applications** folder and start **Habit Flow**.

---

## ⚠️ "Developer Cannot Be Verified" Warning? (Gatekeeper Bypass)

Because this app is free and distributed directly outside the official Mac App Store, macOS will block it on the first launch with a warning saying the developer is unverified or the app is damaged. 

To open it safely **without running any scripts**:

1. Open your **Applications** folder.
2. **Right-Click** (or hold the `Control` key and click) on the **Habit Flow** icon.
3. Select **Open** from the context menu.
4. A prompt will appear with a third option. Click the **Open** button to launch the app.

> [!NOTE]
> You only need to do this **once**. macOS will remember your decision, and in the future, you can launch Habit Flow by double-clicking it normally.
